# Act 0 Action Literacy

- Fold means leave now and save chips.
- Check means continue for free when no bet is in front of you.
- Bet means put chips in first when no bet is in front of you.
- Call means match the current bet and keep playing.
- Raise means increase the current bet.
- If toCall == 0, check can be available and bet starts the action.
- If toCall > 0, call matches and raise increases.
- Good folds are wins when pressure is bad.
- Win condition: pick the legal action with calm timing.
