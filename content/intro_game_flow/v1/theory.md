# Street Flow Injection

A hand follows a fixed order:

- Preflop: private cards, first decisions.
- Flop: three community cards.
- Turn: fourth community card.
- River: fifth community card.

Use one table anchor before speed:
- Button and blinds define orientation.
- If orientation is stable, street order decisions stay stable.

This module trains one deterministic anchor first, then repeats it in isomorphic reps.
