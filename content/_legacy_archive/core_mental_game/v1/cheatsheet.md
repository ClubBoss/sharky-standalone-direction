# Cheatsheet

## Key Rules
- Run a pre-session checklist (sleep, food, hydration, goal, time box, stop-loss, end-on-time, A-game cue).
- Define stop-loss/end-on-time caps to avoid chase tilt and late-session decay.
- Use fast interrupts (breathe-reset, stand/walk) to lower arousal without turning the session into a break.
- Tag problem hands for later review but do not relive them mid-session.
- Protect attention by reducing tables, silencing notifications, and moving away from toxic tables.
- Normalize swings: expect downswings/heaters so emotions stay steady.

## Fast Heuristics
- A 10-second breathe-reset works better than a deep dive mid-session.
- Cutting tables when tired keeps reads sharp.

## Quick Numbers
- Tilt interrupt: breathe/reset for 10 seconds or walk for 2 minutes.

## Examples (1–2 lines)
- After a bad run on K72 5 2, take 10-second breathe-reset, tag the hand, reduce tables, and hydrate.
