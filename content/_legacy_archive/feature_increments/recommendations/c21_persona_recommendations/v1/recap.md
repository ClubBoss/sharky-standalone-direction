### C21 Persona Recommendations Recap

Persona-aware recommendations keep study aligned with how you learn and how you feel. Start by reading persona traits: preferred session length, tolerance for complexity, and appetite for aggression. Then fold in Tier-A emotion states: calm can handle deeper drills and longer blocks; focus pairs well with timed micro-quizzes; tension calls for short, confidence-building sets. Every action is tagged with position (UTG, MP, CO, BTN, SB, BB), pot type (SRP, 3BP), and board class so the system knows which patterns to repeat.

Performance trends drive difficulty. Accuracy above 85% unlocks harder spots: 3BP OOP dynamic boards, bluff-catch rivers versus overbluff tags, and thin value versus overcallers. Accuracy between 70-85% holds mixed difficulty with hints. Below 70%, the engine serves foundational micro-quizzes and simple drills to rebuild streaks. Spaced repetition windows adapt to results: misses drop back to early windows, clean passes stretch forward.

The system pulls history from C13 through C20. C13 exploit triggers reveal which conversions you miss. C14-C16 checkpoint work shows whether you handle multi-street shifts. C17 recaps measure cross-spot consistency. C18 and C19 record speed and recall on dense prompts. C20 logs spacing success and recurring tags. Combined, they form a persona-tuned queue that stays deterministic but responsive.

Leak tags connect emotion and content. Overfold issues schedule more double-barrel prompts; overcall issues schedule thin value reps; overbluff flags generate bluff-catch practice; undersized cbet tags call for raise or float reps. Session-length heuristics keep engagement high: 10-15 minute blocks for tension personas, 20-30 for calm grinders, and short sprints for focus-first players.

Live and online paths diverge slightly. Live volume is lower and emotions swing more, so reviews come sooner after play and stay shorter. Online grinders can insert in-session micro blocks and follow the standard spacing ladder. Contrast with static plans: a fixed script ignores mood and performance, leading to boredom or overload. Persona-aware recommendations stay adaptive but predictable, selecting the next item based on persona traits, Tier-A emotion, recent accuracy, leak tags, and spacing window. The result is a stable loop that keeps poker heuristics ready under pressure.
