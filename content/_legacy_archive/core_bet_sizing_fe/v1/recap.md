# Recap

## Key Ideas
- Every size has a purpose; small/medium suits merge ranges on static textures while large/overbet serves polar lines with dynamic boards or capped opponents.
- Pair size with range shape: polar ranges need big/overbet sizes, merge ranges rely on small/medium bets, and keeping frequencies coherent avoids leaking value.

## Common Mistakes
- Using one size for all textures, which either gives draws good odds or fails to build pots with better hands.
- Bluffing large with merge ranges, leading to folds from worse or calls from better.

## Quick Checks
- When should you switch from small to medium sizing on the turn?
- How do polar and merge ranges choose their default bets?
