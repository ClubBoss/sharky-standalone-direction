# What it is
Advanced range synthesis merges advantage reads, density, layering, weighting, pressure mapping, reaction layers, and exploit windows so every signal feeds one decision map.

# Why it matters
With synthesis you classify ranges cleanly, attack only when fold equity exists, and keep thin-value windows from turning reckless.

# Rules of thumb
- Start with the top-end and density backbone before adding bluffs.
- Map the pressure tier so you know how much aggression the node tolerates.
- Align layering with the pressure response to protect fragile regions.
- Adjust thin-value only when the reaction layer stays steady.
- Spot exploit windows by matching density gaps to fold-heavy tiers.
- Never treat density as advantage; even dense zones can fold.
- Check fragility before ramping bluff weight.
- Keep adjustment loops tight so tweaks reinforce existing signals.

# Mini example
Attacker shows heavier top-end and robust mid-density while the defender stays semi-capped. Synthesis reads mid/high pressure, keeps aggression measured, and waits for a thin-value or timing window before extending bluffs.

# Common mistakes
- Overweighting density gaps and mistaking them for advantage.
- Treating pressure tiers as fixed instead of watching them shift.
- Labeling thin-value efforts as stabs when patience is required.
- Ignoring reaction shifts and forcing one line regardless of cues.
- Letting advantage cues override fragility signals and losing leverage clarity.
- Seeing windows where pressure is low and folding to any resistance.
- Forcing symmetry in asymmetrical nodes, which breaks alignment.
- Skipping adjustment loops after a window decays, so stale plays persist.

# Workflow
- Identify the advantage backbone and its top-end carriers.
- Assess density and layering so each segment holds a role.
- Map the pressure tier to understand acceptable aggression.
- Read the reaction layer for calls, folds, or raises.
- Classify windows (thin-value, timing, fold, or stab) from those inputs.
- Choose conservative or assertive lines based on fragility and tier behavior.

# Mini-glossary
- synthesis : the unified model linking every module.
- density backbone : the concentrated combos that supply leverage.
- pressure tier : the aggression level guiding sizing.
- reaction layer : the defender’s dominant response mix.
- thin-value line : subtle pressure when fold equity exists.
- timing window : a short-lived pocket for controlled aggression.
- fragility zone : segments that collapse under pressure.
- exploit seed : the earliest moment a leak becomes actionable.
- adjustment loop : the cycle that recalibrates after each change.
- layer alignment : matching layers to pressure and reactions.

# EV
Lean synthesis shrinks EV leakage by keeping decisions tied to layered signals. It prevents over-stabs in fragile nodes and keeps thin-value/timing windows reliable.
