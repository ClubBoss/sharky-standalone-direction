# Hand Ranking Anchor

At showdown, the pot goes to the best five-card hand.

Use this short mental ladder:
- Pair < Two Pair < Three of a Kind < Straight < Flush < Full House < Four of a Kind < Straight Flush.

Two practical rules:
- Category first, kicker second.
- If categories differ, higher category always wins.

Do not solve everything at once.
In early training, first identify category correctly, then compare kickers only when categories tie.

This module uses a short isomorphic ladder to repeat category recognition.
