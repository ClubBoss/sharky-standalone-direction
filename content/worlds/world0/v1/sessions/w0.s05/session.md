# Session w0.s05

## Objective
Practice repeat reps for stable street order reading and seat recognition without falling back to guessing.

## Scenario
Each rep revisits the same World 0 foundations with slightly different anchors:
dealer button seat,
nearby seats,
the current board street,
and the action buttons that belong to the spot.
The goal is to make table-reading automatic enough that the learner can keep the order of the hand clear under repetition.

## Decision
Choose the answer that best matches the current seat,
street,
or action cue in the rep.

## Explanation
World 0 needs repetition, not strategy complexity.
This session keeps the learner focused on the table layout and the order of the hand so that later action choices start from clear table literacy instead of intuition.
