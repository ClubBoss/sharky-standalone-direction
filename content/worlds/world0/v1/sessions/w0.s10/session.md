# Session w0.s10

## Objective
Run the final World 0 checkpoint by combining seat recognition, acting seat tracking, action buttons, and street order awareness.

## Scenario
Each rep revisits a compact table state and asks the learner to identify where the action is,
whose turn it is,
what the visible action buttons mean,
and which street the hand is on.
The goal is to read the table state cleanly before any action choice is imagined.

## Decision
Choose the answer that best matches the current seat,
acting seat,
action button,
and street order cues in the spot.

## Explanation
This checkpoint closes World 0 by testing whether the learner can keep the core table-literacy habits together.
The goal is not strategy,
only a stable learner-facing habit of reading seats,
turn order,
action names,
and street order before moving on to richer decision work.
