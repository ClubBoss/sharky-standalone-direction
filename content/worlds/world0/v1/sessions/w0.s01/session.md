# Session w0.s01

## Objective
Build the first World 0 orientation habit by locating the blinds, reading the visible cards, and recognizing the simplest action buttons.

## Scenario
Each rep shows a compact table state and asks the learner to confirm the most basic anchors first:
where the small blind and big blind are,
which visible cards belong to the learner,
and what the basic action buttons mean.
The goal is not strategy,
only clean table orientation before richer decision work begins.

## Decision
Choose the answer that best matches the visible seat,
card,
board,
or action cue in the spot.

## Explanation
World 0 starts by removing guesswork.
Learners should first become comfortable finding the blinds,
reading simple table anchors,
and recognizing basic actions before they are asked to think about why a poker decision is stronger.
