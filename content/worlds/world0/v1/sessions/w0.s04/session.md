# Session w0.s04

## Objective
Rehearse the blind-side layout until the nearby seats, basic action buttons, and turn card remain stable under repetition.

## Scenario
Each rep returns to the same compact table read on purpose.
The learner restarts from the small blind seat,
checks seat S0,
and repeats fold, call, raise, and the turn card until the pattern feels automatic.
The goal is to remove hesitation from the first repeat set.

## Decision
Choose the answer that best matches the repeated seat,
street,
or action button cue in the rep.

## Explanation
World 0 needs repetition before variation.
This session turns the blind-side layout into a dependable habit so later seat shifts do not break the learner's table-reading foundation.
