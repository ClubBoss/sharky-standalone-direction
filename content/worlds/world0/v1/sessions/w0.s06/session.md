# Session w0.s06

## Objective
Run a mixed World 0 checkpoint by combining seat recognition, action buttons, and street order awareness.

## Scenario
Each rep revisits a compact table state and asks the learner to identify the seat cues,
match the visible action buttons to their meaning,
and track which street the hand is currently on.
The goal is to hold the table-reading basics together in one mixed check.

## Decision
Choose the answer that best matches the seat,
action button,
and street order cues in the spot.

## Explanation
This checkpoint reinforces that World 0 is about reading the table correctly before any strategy begins.
The learner practices keeping seats,
action names,
and street order stable together so later worlds can build on reliable table literacy.
