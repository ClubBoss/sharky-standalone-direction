# Session w0.s02

## Objective
Extend the first World 0 table read beyond the blind anchors by locating the dealer button seat, both hole cards, and the next board cards in order.

## Scenario
Each rep starts from the blind anchors and then pushes one step farther across the table.
The learner confirms seat S1,
the dealer button seat,
both hole cards,
and the middle flop-to-turn path without guessing.
The goal is to widen the table layout cleanly before action names are chosen.

## Decision
Choose the answer that best matches the visible seat,
card,
or action button cue in the spot.

## Explanation
World 0 is still about orientation, not strategy.
This session teaches the learner to keep the table layout, hole cards, and early board order stable while the action buttons stay visible.
