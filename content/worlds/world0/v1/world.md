# World 0

## Overview
Cognitive shift: stop guessing at the table and start reading the layout and action order with confidence.
You move from "I click something" to "I know whose turn it is and what each action means."
Not yet part of this world: hand-strength debates, bluff-catching, bluffing stories, or bet-sizing tradeoffs.
World 0 is only the first clean table-reading pass.

## Goals
- Practice seat recognition and blind order until it is automatic.
- Practice action buttons (fold, call, raise, check) in clear, repeated spots.
- Practice street order: preflop, flop, turn, river.
- Practice spotting the acting seat before choosing an action.

## Completion criteria
- You can identify seat/action/street cues quickly without hesitation.
- Common mistake we fix: acting from intuition before confirming turn order.

## Release-grade finish proof
- The learner can name blind anchors, the acting seat, street order, and basic action buttons without guessing.
- Mixed checkpoints confirm the same table-reading skill under repetition and speed instead of slipping into strategy drift.

## Handoff to World 1
- World 1 begins only after this table-reading layer is automatic and stable.
- The next world can add stronger live learner decisions because World 0 now closes the table-reading foundation cleanly.
