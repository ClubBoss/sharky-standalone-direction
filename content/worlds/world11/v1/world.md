# World 11: Real Play Transfer

## Overview
- Goal: carry earlier table-reading skills into one calm, repeatable real-play process.
- Cognitive shift: knowing a concept and using it during a live hand are different skills; a short process keeps the next decision clear.

## Goals
- Set one observable focus before a session starts.
- Notice position, a visible player tendency, board texture, and price before acting.
- Name one action objective using earlier value, bluff, protection, or fold language.
- Record one useful observation for the next review.

## Boundaries
- This source draft reinforces earlier World 1-10 skills; it does not introduce solver lines, specialist strategy, or a complete mindset curriculum.
- It is authored source only. It does not register a campaign, learner entry, completion state, or progression handoff.
