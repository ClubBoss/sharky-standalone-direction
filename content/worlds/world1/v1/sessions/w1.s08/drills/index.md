# Drills
- choose_small_blind_fold_oop_focus_v1: small blind focus rep where out-of-position weakness should release instead of drifting into a loose continue.
- choose_big_blind_call_oop_defend_focus_v1: big blind focus rep where a playable hand is only clean enough for a restrained continue by calling.
- choose_small_blind_raise_oop_clean_start_v1: small blind focus rep where acting first still allows a raise only when the start is clearly strong and clean.
