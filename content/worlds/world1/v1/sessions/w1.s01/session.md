# Session w1.s01

## Objective
Build the first World 1 bridge by making position-first choices from clean preflop starts.

## Scenario
Each rep shows a simple preflop spot and asks whether position should make the hand easier to continue or easier to release.
The goal is to notice that the same-looking hand can change once seat and action flow are acknowledged first.
The session ends with a short size-label preview so learners can recognize common buttons later, but World 1 is still about orientation first, not full bet-sizing study.

## Decision
Choose the simple preflop action that best fits the seat and position in the spot.

## Explanation
World 1 starts when learners stop clicking actions in isolation.
Position changes how safely a hand can continue,
so the first habit is to read where you are seated before deciding whether a start is clean enough to play.
A few closing reps introduce common size labels only as vocabulary preview.
The full mainline home for bet-sizing purpose and price tradeoffs is World 4.
