# Drills
- choose_raise_when_action_folds_to_you_focus_v1: first-in focus rep where folded action leaves the cleanest raise from a comfortable seat.
- choose_call_when_open_reaches_you_focus_v1: facing-action focus rep where seat comfort still leaves a clean continue by calling, but only when the hand is strong enough not to drift into trouble.
- choose_fold_when_pressure_and_position_fail_focus_v1: facing-action focus rep where pressure plus a tempting but weak-looking continue should be released before trouble grows.
