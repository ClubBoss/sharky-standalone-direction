# Session w1.s07

## Objective
Reinforce in-position preflop responses as the start of the focused World 1 tail.

## Scenario
Each rep puts the learner in late position and asks for the simplest clean response once seat advantage is clear.
The goal is to make in-position preflop choices feel easier to continue with when the seat gives cleaner control.

## Decision
Choose the action that best fits the clean in-position response in the spot.

## Explanation
This session starts the `w1.s07-w1.s09` focused tail.
The learner should notice when position gives them the easier continue,
then use that seat advantage to choose a cleaner preflop action without forcing a loose guess.
Position does not make every hand playable,
but it does make the honest continue feel more comfortable when the hand is already clean enough.
