# Session w6.s03

## Objective
Build the first World 6 checkpoint by noticing early flop range advantage in plain language.

## Scenario
Each rep shows a simple flop and two likely ranges.
The learner decides which side the board helps more and whether that side should feel more comfortable applying pressure.

## Decision
Choose which range has the clearer early advantage on the flop.

## Explanation
This checkpoint turns range buckets and width into a first practical board question:
which side connects with the flop more naturally.
The goal is not solver-grade precision, only a bounded read on which range is stronger or more comfortable on the board.
