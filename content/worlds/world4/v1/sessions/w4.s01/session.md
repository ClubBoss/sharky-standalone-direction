# Session w4.s01

## Objective
Teach the first stable bet-purpose cluster by matching a size preset to what it is trying to accomplish.

## Scenario
Each rep presents one compact sizing purpose:
keep weaker hands in,
keep the price easy,
apply maximum pressure,
or reopen action without bloating the pot.
The learner succeeds by connecting the preset to the purpose instead of treating size as decoration.

## Decision
Choose the preset that best matches the betting purpose in the spot.

## Explanation
World 4 begins by turning the proven sizing pilot into a mainline competency.
Half pot, one third pot, pot, and minimum raise each do a different job.
The first pass stays compact on purpose: learn the purpose first, then build deeper price and board reasoning later.
