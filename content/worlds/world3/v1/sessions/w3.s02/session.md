# Session w3.s02

## Objective
Continue the preflop framework by reusing simple hand categories and checking which compact open / call / fold choice fits once the spot changes.

## Scenario
You stay in plain-language preflop situations. Each rep keeps the framework small: identify the hand bucket, notice the basic context, and choose the most sensible first action without charts or advanced policy trees.

## Decision
Use the hand category first, then decide whether the spot calls for an open, a call, or a fold.

## Explanation
This is the compact open / call / fold continuation for the World 3 front slice. Framework learning improves through repetition with small context changes, so the learner builds a stable preflop map before deeper sizing, range, board-texture, and specialization layers appear.
