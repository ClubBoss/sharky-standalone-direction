# Session w3.s12

## Objective
Build the next World 3 preflop bridge by learning when position helps hero continue against an opener and when the hand is still too weak to keep going.

## Scenario
The same authored preflop table frame stays in view while the learner first reads who has position, then decides whether a stronger suited broadway hand should continue versus a cutoff open, then decides whether a weaker offsuit hand should simply fold in that same frame.

## Decision
Read position first, then let hand quality and the action frame decide whether continuing still makes sense.

## Explanation
World 3 gets more useful once position stops acting like a permission slip for every hand. Position matters, but the hand still has to be strong enough for the frame.
