# Session w3.s04

## Objective
Start the World 3 continuation block by repeating premium and strong-hand category reps until the first preflop action feels stable.

## Scenario
You stay in plain-language preflop spots with stronger hand buckets. This continuation block keeps the framework learner-facing: each rep asks whether the cleaner first action is open, call, or fold once the frame changes.

## Decision
Start with the hand category, then choose the most disciplined compact preflop action.

## Explanation
This is the premium and strong-hand stability opener for the World 3 continuation cluster. World 3 gets more usable when stronger buckets stop feeling random, so repeating a few stable raise, call, and fold patterns builds trust before medium-hand discipline and the mixed checkpoint arrive.
