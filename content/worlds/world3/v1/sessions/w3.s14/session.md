# Session w3.s14

## Objective
Close the World 3 live bridge by showing that the same middling hand can open in late position but still fold from an earlier seat in the same unopened-pot frame.

## Scenario
The same authored preflop table frame stays in view while the learner first reads who has position, then decides whether the middling hand is good enough to open late, then sees that the same hand should still fold when the seat advantage shrinks.

## Decision
Read position first, then let the seat advantage decide whether the same middling hand wants raise or fold in the unopened-pot frame.

## Explanation
World 3 closes when hand category and position work together instead of competing. The same middling hand does not carry one universal answer: later position can justify the open, while an earlier seat still asks for discipline and a fold.
