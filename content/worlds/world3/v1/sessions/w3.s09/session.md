# Session w3.s09

## Objective
Continue the final World 3 tail by training category-first decisions when similar hands need different first actions.

## Scenario
You revisit similar preflop hands in slightly different frames. This final World 3 block asks whether the same broad category now wants an open, a call, or a fold.

## Decision
Start with the hand category, then choose the compact preflop action that fits the position and action frame.

## Explanation
This is the same-hand different-action continuation for the final World 3 tail. It sharpens one of the key World 3 habits: the same hand family can deserve a different first action once the frame changes, and the learner should be able to explain why in plain language.
