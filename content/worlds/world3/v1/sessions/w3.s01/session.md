# Session w3.s01

## Objective
Open World 3 by sorting hands into simple categories and matching them to one compact open / call / fold choice.

## Scenario
You start with plain-language preflop spots. Each rep asks for one stable first decision based on hand category and basic table context, without charts or advanced tree memorization.

## Decision
Identify the hand bucket first, then choose the simplest fitting preflop action: open, call, or fold.

## Explanation
This is the hand-categories opener for World 3. Preflop framework starts by reducing chaos:
stable categories and one-reason action rules stop random first decisions and prepare you for deeper sizing, board-texture, and range work later.
