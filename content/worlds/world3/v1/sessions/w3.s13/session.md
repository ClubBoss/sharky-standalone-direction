# Session w3.s13

## Objective
Build the next World 3 preflop bridge by learning that position helps a playable hand open, but does not rescue a clearly weaker hand in the same unopened frame.

## Scenario
The same authored preflop table frame stays in view while the learner first reads who has position, then decides whether a stronger in-position hand should open in an unopened pot, then decides whether a weaker hand should simply fold in that same frame.

## Decision
Read position first, then let hand quality decide whether the unopened-pot frame wants raise or fold.

## Explanation
World 3 gets more practical once the learner sees that position is an advantage, not a free pass. In unopened pots, better in-position hands open and weaker hands still let the spot go.
