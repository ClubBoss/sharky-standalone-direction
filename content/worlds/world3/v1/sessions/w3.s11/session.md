# Session w3.s11

## Objective
Build the first World 3 preflop decision frame by connecting position to an unopened-pot choice and then to a facing-open response.

## Scenario
The same authored preflop table frame stays in view while the learner first reads who has position, then uses one broadway hand to decide how to act in an unopened pot, then uses that same hand again when facing an opener.

## Decision
Read the seat advantage first, then let the action frame decide whether the hand wants raise, call, or fold.

## Explanation
World 3 starts when position stops being trivia and starts shaping preflop choices. The same hand can raise in one frame and call in another because the action context changed.
