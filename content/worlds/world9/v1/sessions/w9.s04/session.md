# Session w9.s04

## Objective
Practice aggression-frequency control when an exploit spot invites pressure but overdoing it would become careless.

## Scenario
Each rep shows a player tendency that creates room for more pressure, but not unlimited pressure.
The learner decides when an exploit should become a measured increase in aggression rather than an automatic maximum-pressure line.

## Decision
Choose whether the exploit spot calls for restrained aggression, stronger pressure, or a return to baseline discipline.

## Explanation
Exploit play is not just about finding permission to attack.
Aggression-frequency control matters because a good exploit can become a bad habit if the learner stops checking whether the pressure still fits the actual tendency.
