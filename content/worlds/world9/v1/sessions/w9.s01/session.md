# Session w9.s01

## Objective
Build the first World 9 bridge by separating a balanced default from an exploit opportunity.

## Scenario
Each rep shows a simple opponent tendency and asks whether the spot should still be played as a standard baseline or whether a practical exploit is already justified.
The goal is to notice that real-player adjustments start only when a tendency is clear enough to matter.

## Decision
Choose whether the spot still calls for a baseline line or whether a bounded exploit adjustment is justified.

## Explanation
World 9 starts when learners stop assuming every opponent behaves like an ideal model.
Exploit baseline thinking means recognizing when to stay disciplined and when a visible player tendency gives permission to adjust.
