# Session w5.s11

Practice basic outs awareness by counting the most common beginner draw shapes before any deeper action policy.

You stay in compact board-reading spots. Each rep asks you to identify whether the draw is a flush draw, open-ended straight draw, or gutshot, then count only the straightforward improvement cards.

Read the draw first: a flush draw has 9 outs, an open-ended straight draw has 8 outs, and a gutshot has 4 outs.

Outs are a foundation counting tool, not a full strategy. World 5 uses them only to make board improvement visible before later strategy work.
