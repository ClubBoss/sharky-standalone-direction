0. PURPOSE

Этот документ — единый источник правды (SSOT) для всей системы обучения Poker Analyzer.
Он описывает:

• все модули (core → cash → mtt → hu → live → online → solver → evolution → persona)
• строгий порядок прохождения
• уровни прогресса (Entry → Mid → Advanced → Mastery → Final Trajectory)
• связи между модулями
• зависимости и prerequisites
• EV-приоритеты
• структура генерации контента (theory → drills → demos → quiz → recap → checkpoint → evolution)
• точную дорожную карту для генерации и QA

На него будет опираться генерация контента Codex/ChatGPT и проверка качества.
