# Manual Legacy Postflop Slice

This folder is `transitional runtime content`.

- It is still actively consumed by learning-path and legacy launcher flows.
- It is not canonical authored-module content for the current product direction.
- Do not treat files in this folder as proof of canonical World1 module identity.
- Future work should either:
  - replace a file with authored-module content under `content/`, or
  - keep it explicitly as long-term legacy runtime content by decision.

Until one of those happens, this slice is `legacy-but-needed`, not removable and not canonical.
