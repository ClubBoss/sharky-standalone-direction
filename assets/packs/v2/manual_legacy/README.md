# Manual Legacy Runtime Shelf

This subtree is transitional runtime content.

- It is still actively consumed by current learning-path and legacy launcher flows.
- It is not canonical authored-module content for the current product direction.
- Files under this subtree must not be treated as proof of canonical World1 module identity.
- Keep local subfolder classifications where they add useful detail.

Future work should either:

- replace a slice with authored-module content under `content/`, or
- retain that slice explicitly as long-term legacy runtime content by decision.

Until one of those happens, `assets/packs/v2/manual_legacy/*` is legacy-but-needed, not removable, and not canonical.
