# Layout Guides

## Grid
- Base grid: 12 columns
- Gutter: 24 px
- Max content width: 1280 px
- Responsive strategy: clamp sections to 960 px on tablet, stack vertically on mobile

## Spacing Scale
- XS: 8 px
- SM: 16 px
- MD: 24 px
- LG: 32 px
- XL: 48 px

## Radius
- Primary corners: 12 px (cards, modals)
- Secondary corners: 16 px (action badges)
- Pills: 999 px (full stadium)

## Shadows
```
Elevation 1: 0 1 2 0 rgba(0,0,0,0.24)
Elevation 2: 0 4 8 0 rgba(0,0,0,0.28)
HUD Toast: 0 12 32 0 rgba(0,0,0,0.35)
```

## Typography Pairing
- H1/H3: AppTypography.h1 / h3 (18/16, weight 600)
- Body: AppTypography.body (16, normal)
- Label: AppTypography.label (14, medium)
- Caption: AppTypography.caption (12, regular)
