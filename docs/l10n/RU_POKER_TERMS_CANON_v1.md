# RU POKER TERMS CANON v1

Status: ACTIVE  
Purpose: human-quality Russian terminology and tone canon for the launch-facing
Sharky product.

## Goal

Russian in the app must read like a real product written for Russian-speaking
players, not like a direct English mirror.

This canon exists to keep launch-facing copy:

- clear
- warm
- poker-literate
- compact
- consistent across `Placement`, `Home`, `Learn`, `Play`, `Review`, `You`,
  `Table`, and `Result`

## Tone

The product voice in Russian should be:

- calm
- confident
- modern
- human
- supportive

The product must not sound:

- machine-translated
- bureaucratic
- over-hyped
- childish
- solver-heavy
- like an admin tool

## Core language rules

1. Prefer short natural Russian over literal English structure.
2. Keep one poker idea per line.
3. If a phrase sounds translated, rewrite it completely instead of polishing it.
4. Use Russian for product actions and navigation.
5. Keep advanced poker jargon only where it is already normal for the target
   player.
6. Do not expose internal product language in learner-facing Russian.

## Product wording rules

- `Home` should sound like a clear next step, not a dashboard.
- `Learn` should sound like a path, not a catalog.
- `Review` should sound like repair, not punishment.
- `Play` should sound optional and light.
- `You` should sound like growth and identity, not analytics.
- `Result` should sound like closure plus next move.
- `Sharky` should sound warm, short, and context-aware.

## Preferred product vocabulary

- `Continue lesson` -> `Продолжить урок`
- `Quick practice` -> `Быстрая практика`
- `Fix next` -> `Исправить дальше`
- `Open Review` -> `Открыть разбор`
- `Keep going free` -> `Продолжить бесплатно`
- `Not now` -> `Не сейчас`
- `Locked` -> `Закрыто`
- `Coming soon` -> `Скоро`
- `Streak` -> `Серия`
- `Milestones` -> `Вехи`
- `Consistency` -> `Ритм` or `Постоянство` depending on context

## Poker terminology policy

Use standard poker vocabulary when it is more natural than forced translation.

Keep as accepted poker borrowings when context is learner-safe:

- `префлоп`
- `постфлоп`
- `дро`
- `рейз`
- `колл`
- `фолд`
- `блайнды`
- `стрик` only in very light product contexts; prefer `серия` in key surfaces

Prefer Russian explanation around the term instead of translating the term
itself into awkward wording.

Examples:

- good: `На префлопе ты уже понимаешь порядок действий.`
- bad: `До флопа ты понимаешь порядок действий.` when it sounds less natural in
  the product context

## Avoid

- `оптимальная линия`
- `солвер`
- `ветка решения`
- `апгрейд навыка`
- `контекст стабилен`
- `верифицируй давление`
- `прогресс сохранён успешно`

Rewrite into normal learner-facing Russian.

## Sharky voice

Sharky in Russian should be:

- short
- warm
- slightly playful
- never verbose

Good:

- `Три дня подряд. Один чистый реп — и серия жива.`
- `Сначала починим это место. Дальше пойдёт легче.`
- `Хорошо. Маршрут держится.`

Bad:

- `Ваш текущий прогресс стабилен и будет сохранён.`
- `Рекомендуется выполнить корректирующую сессию.`

## Launch priority

Human-quality Russian must land in this order:

1. `Placement`
2. `Home`
3. `Learn`
4. `Review`
5. `Play`
6. `You`
7. `Table`
8. `Result`
9. `Premium preview`
10. Settings / legal / supporting surfaces

## Non-goal

This canon does not require immediate full-repo localization.

It governs the active launch path first.
