## 6. Store/public copy draft

Drafts only. Not submission copy.

### App subtitle / short tagline

The table coach. Learn to read the table from your first hand.

### Short description

Sharky teaches poker from the table up: spot one clue, make one decision, see why it worked, and practice the weak spot right away.

### Screenshot captions

1. Read the table before choosing.
2. Get a clear why after each answer.
3. Practice the same clue while it is fresh.
4. See your local proof at session close.
5. Continue through the W1-W4 Foundation path.

### Feature bullets

- Beginner-safe poker training built around real table clues.
- Short lessons with one decision at a time.
- Feedback explains the clue behind the choice.
- Repair reps repeat the same signal, not random quizzes.
- W1-W4 Foundation path: table basics, hand discipline, position, and preflop framework.

### Claim-safe long description draft

Sharky is a table-first poker coach for players learning to make cleaner decisions from the first hand.

The public v1 Foundation teaches you to read the table in small steps: notice the clue, choose once, understand why, and practice the weak spot while it is fresh. The first four worlds cover table basics, hand discipline, position thinking, and a simple preflop framework.

Sharky is designed around a longer 36-world Core Shark Path, but public v1 represents the W1-W4 Foundation beachhead. Later premium-depth worlds and advanced systems should be described only when they are deliberately activated and claim-safe.
