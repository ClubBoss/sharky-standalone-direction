## 4. W3 Title/Job Truth

The active learner-facing route label is:

`Position Thinking`

That title implies a learner job centered on position as the first explanatory
signal:

- read who acts later;
- understand how position changes information and action quality;
- let position shape simple preflop raise/call/fold decisions;
- explain why the same hand can change action when seat or prior action
  changes.

The current W3 source file describes a broader and different job:

`preflop framework bridge`

That source job teaches hand categories, compact open/call/fold logic, and
one-reason preflop decisions. Position appears inside that job, but it is not
the full owner of the source. The honest route truth is therefore mixed:

- W3 has enough Position Thinking source for two bounded canonical families.
- W3 also contains preflop-framework bridge material that should not be counted
  as new Position Thinking breadth by metadata only.
