## 8. Cascade Mechanism and Proof

The offset cascade is confirmed by a mutually consistent prerequisite chain
across W4-W9 world.md files. Each world's source correctly assumes the
previous world's source content as prerequisite, while the route titles are
shifted one world forward:

```
W5 source: "comes after World 4 because purpose and price need to exist first."
  -> W5 source assumes W4 = Bet Purpose / Price
  -> W4 route = "Preflop Framework" (not Bet Purpose / Price)
  -> W5 source creates a paradox: it requires W5-level content as its own prerequisite

W6 source: "comes after board texture, draws, and improvement counting."
  -> W6 source assumes W5 = Board Awareness
  -> W5 source IS board awareness; consistent in cascade reading

W7 source: "comes after board reading and range-thinking basics."
  -> W7 source assumes W6 = Range Thinking
  -> W6 source IS range thinking; consistent in cascade reading

W8 source: "comes after board reading, range thinking, and stack-depth basics."
  -> W8 source assumes W7 = Stack Depth
  -> W7 source IS stack depth; consistent in cascade reading

W9 source: "comes after board reading, range thinking, stack-depth reasoning,
           and tournament-pressure basics."
  -> W9 source assumes W8 = Tournament/ICM
  -> W8 source IS tournament/ICM; consistent in cascade reading
```

The prerequisite chain is internally coherent when read against the SOURCE
content sequence but contradicts the ROUTE TITLE sequence. This confirms that
source was authored for a correctly sequenced curriculum, while the route titles
were assigned one world too early across W4-W9.

After normalization the cascade resolves cleanly:

```
W4 normalized: "Bet Purpose / Price"     (matches W4 source)
W5 normalized: "Board Awareness"          (matches W5 source; resolves W5 paradox)
W6 normalized: "Range Thinking"           (matches W6 source)
W7 normalized: "Stack Depth"              (matches W7 source; for future unlock)
W8 normalized: "Tournament / ICM"         (matches W8 source; for future unlock)
W9 normalized: "Exploit Thinking"         (matches W9 source; for future unlock)
```

The displaced "Preflop Framework" concept has a natural home: W3 source
(world.md: "World 3 is the preflop framework bridge"). W3 bridge content stays
bridge-limited. No W3 reopening is needed.
