# Today Spine Copy Reconciliation Packet v1

Status: ACTIVE
Last updated: 2026-05-06

## Purpose

Reconcile older content-system phrasing with the current calmer learner-facing
product tone.

This packet is intentionally tiny.
It exists to close the last meaningful copy-policy residue without reopening
broader product or content-governance work.

## Core Rule

Today copy should feel:

1. clear
2. calm
3. useful
4. human

It should not feel:

1. hype-heavy
2. slogan-like
3. mascot-overbearing
4. more intense than the actual product experience

## Replacement Decisions

Use these wording directions in active content truth:

| Older phrasing direction | Replace with |
| --- | --- |
| `Today you forge: [Skill Focus]` | `Today: [Skill Focus]` or `Focus today: [Skill Focus]` |
| `you are becoming a shark` as direct identity slogan | calm identity reinforcement tied to visible progress, not mascot mythology |
| strong motivational headers that sound louder than the product | short, practical, learner-safe phrasing |

## Tone Standard

Preferred Today language:

- "Today: Position decisions"
- "Focus today: Spot the price"
- "Repair this weak spot first"
- "Run your daily set"

Avoid:

- "Today you forge"
- mascot-first slogans as default headers
- grand transformation claims before the learner has felt them

## Why

The current product-working direction is:

1. value first
2. calm trust
3. visible progress
4. return because the app helps

Older louder phrasing weakens that.

The copy should support habit by feeling useful and believable, not by sounding
like a motivational system.

## Scope

This packet only governs:

1. Today/header-style phrasing in active content docs
2. identity-reinforcement wording direction in content-system level docs

It does not reopen:

1. Sharky system scope
2. runtime copy already verified elsewhere
3. broader app UX language audits

## Practical Outcome

After this packet, the content-control plane should treat calmer learner-facing
tone as the default.

That means:

1. short headers
2. progress-linked reinforcement
3. no loud slogan pressure

This is the final copy-policy cleanup needed before returning fully to product
work.
