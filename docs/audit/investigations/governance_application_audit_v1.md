## Selection Result

- selected next pilot compliance block:
  - World 2 surfaced bridge-path governance compliance pilot

## Why This Block Wins

It dominates on the requested criteria:

- bounded scope
  - one implemented world-scale bridge segment rather than the whole curriculum
- direct product EV
  - World 2 is the strongest current “next ladder” after World 1 and therefore the highest-risk place for path-shape drift
- compatibility with current implemented surfaces
  - the sessions already exist, the truth/validation work already exists, and the path is already visible in repo truth
- low rewrite risk
  - the likely work is governance/status/pacing/release-gate alignment of an already-bounded bridge, not building new worlds or rewriting the product loop

## What The Pilot Should Check Later

This selected pilot should later audit World 2 as one bounded bridge path for:

- pilot-vs-dense-world status honesty
- whether the current World 2 path is presented as a bridge rather than implied full world maturity
- whether the implemented session order still respects prerequisite and pacing truth
- whether the path has enough review / checkpoint / variation support to remain acceptable as a bounded bridge
- whether current surfaced scope matches release-gate truth for a world/session slice

## Why Other Candidates Did Not Win

### World 1 Live Path

- strongest current model, but not the highest-EV first compliance pilot
- likely to broaden into refinement or polish rather than one sharp governance correction

### Worlds 3–10 Shells

- too broad and too underfilled
- governance application would become structural planning, not a bounded compliance pass

### Home / Intake / Today

- meaningful product surfaces, but the main issues there are entry-shell and habit-loop behavior
- that is adjacent to governance, not the cleanest first curriculum-governance pilot

### Deferred World 2 Heuristic Residues

- still blocked on semantic honesty
- not yet a path-governance compliance block

## Operational Rule

Do not broaden this next step into a full curriculum rewrite.

If this pilot is taken, it should stay bounded to:
- one implemented path segment
- one world-scale bridge
- one governance-alignment pass against the SSOT stack
