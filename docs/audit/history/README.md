# Historical Audit Shelf

This folder contains older standalone audit reports that are still worth preserving but are not active audit work.

Use this bucket for:
- legacy architecture audits
- old surface-specific audit reports
- historical pedagogy or UX audit snapshots
- isolated historical review records that do not justify a separate top-level bucket

For active bounded audit work, use:
- `docs/audit/investigations/`
- `docs/audit/closeouts/`
