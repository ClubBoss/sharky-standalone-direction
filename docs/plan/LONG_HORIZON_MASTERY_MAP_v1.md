# LONG_HORIZON_MASTERY_MAP_v1

## Purpose

Define the first canonical long-horizon mastery truth layer for the learning product so future expansion is governed by one explicit multi-month mastery trajectory instead of local intuition.

This layer governs what durable growth the product should support over time.
It does not replace:
- progression pacing truth
- prerequisite / difficulty truth
- content release-gate truth
- retention / anti-boredom truth
- session energy budget truth

## Governs Now

This SSOT governs:
- what long-horizon progression horizons the product should meaningfully support
- what mastery strata should exist over time
- how breadth and depth should expand after foundations
- how stronger learners should continue finding real depth
- how future world/module growth should ladder into longer-term mastery rather than only first-pass exposure

This SSOT does not yet govern:
- monetization timing
- adaptive personalization
- mastery-map UI presentation
- XP/level economies
- runtime mastery scoring formulas
- full RPG/gamification systems

## Core Principle

The product should remain meaningful over months because the learner keeps gaining:
- wider skill coverage
- deeper discrimination
- stronger transfer
- more stable recall
- more realistic chained reasoning

Long-horizon value must come from real learning depth, not synthetic engagement loops.

## Supported Time Horizons

The product should be planned to feel meaningful across these default horizons:

### 0-3 Months

Primary goal:
- build a stable foundational table-reading and decision base

Expected learner growth:
- orientation and anchor fluency
- basic sequence/order understanding
- stable recognition and classifier basics
- compact visible comparison and simple action-choice confidence

Product meaning at this horizon:
- the learner should clearly feel “I understand more than when I started”

### 3-6 Months

Primary goal:
- move from isolated competence into connected application

Expected learner growth:
- stronger action-choice reasoning
- early size/purpose understanding
- short multi-step reasoning
- mixed review across stable families
- repeated resurfacing of earlier skills in richer contexts

Product meaning at this horizon:
- the learner should feel that separate ideas are starting to connect into usable table logic

### 6-12 Months

Primary goal:
- deepen transfer, resilience, and layered judgment

Expected learner growth:
- broader mixed application
- richer pressure-based decision contexts
- later-world variables such as stack depth, tournament pressure, and more advanced tradeoff reasoning
- stronger checkpoint/capstone evidence across multiple families

Product meaning at this horizon:
- the learner should feel that the product is still opening deeper poker layers, not only recycling beginner repetition

## Canonical Mastery Strata

Use these strata as the default long-horizon growth ladder:

1. orientation literacy
2. pattern recognition stability
3. compact decision fluency
4. connected local reasoning
5. mixed transfer competence
6. later-world strategic depth

Interpretation:
- each stratum builds on the earlier one
- later strata should revisit earlier skills under richer pressure, not discard them
- a learner may progress unevenly by family, but the default path should respect this ladder

## Stratum Meanings

### 1. Orientation Literacy

The learner can:
- locate table anchors reliably
- track simple order/sequence logic
- understand what part of the table or hand is being referenced

### 2. Pattern Recognition Stability

The learner can:
- recognize core families consistently
- classify visible situations without excessive friction
- compare simple visible truths with confidence

### 3. Compact Decision Fluency

The learner can:
- make bounded action choices with one clear local reason
- begin understanding purpose behind decisions
- repeat stable decisions across multiple examples

### 4. Connected Local Reasoning

The learner can:
- connect 2-4 linked ideas on one coherent scenario
- use prior recognition/classification in a decision chain
- survive bounded reduced-guidance application

### 5. Mixed Transfer Competence

The learner can:
- apply stable families in mixed or shifted contexts
- recover earlier skills under variation
- perform in recap/checkpoint/capstone settings with real integration

### 6. Later-World Strategic Depth

The learner can:
- keep growing into richer strategic variables after beginner foundations stop being novel
- handle more realistic tradeoffs, constraints, and pressure conditions
- continue to expand through later worlds without the curriculum flattening into shallow repetition

## Breadth vs Depth Rule

Long-horizon growth should expand in this order:

1. enough breadth to prevent brittle one-skill learning
2. enough depth to stabilize each important family
3. mixed transfer once multiple families are stable
4. later-world strategic depth only after the earlier ladder is credible

Practical rule:
- do not chase endless new surface breadth while depth is fragile
- do not overstay in narrow depth so long that the ladder stops opening

## Stronger-Learner Continuation Rule

Stronger learners should keep finding value through:
- deeper world layers
- richer transfer/capstone work
- more realistic pressure variables
- broader cross-family integration

They should not rely on:
- inflated repetition counts
- cosmetic progression only
- generic “advanced” labeling without a real new cognitive job

## Later-World Expansion Rule

Later worlds should justify themselves by adding one of:
- a genuinely new strategic variable
- a richer transfer context
- a stronger pressure/tradeoff condition
- a deeper application layer built on stable earlier families

A later world should not exist only to restate earlier worlds in thinner or noisier form.

## Mastery Preservation Rule

Long-horizon mastery must remain cumulative:
- earlier families should resurface later
- later growth should prove retention, not assume it
- depth should come from recombination, pressure, and transfer, not only from first exposure to new labels

If a later layer cannot show what prior stable mastery it builds on, it is not a real mastery layer.

## Horizon Decision Table

| Horizon | Main learner need | Product obligation |
| --- | --- | --- |
| 0-3 months | foundations that become stable and usable | provide clear literacy, recognition, and compact decision families |
| 3-6 months | connected reasoning and transfer growth | provide chaining, mixed review, and stronger applied layers |
| 6-12 months | deeper strategic expansion without flattening | provide later-world variables, richer capstones, and broader transfer depth |

## Governing Use Rule

Before adding a future world/module/session expansion, ask:

1. Which mastery stratum does this strengthen?
2. Does it add breadth, depth, or transfer in a justified order?
3. Which long-horizon learner need does it serve: 0-3, 3-6, or 6-12 months?
4. Does it help stronger learners find genuine depth after foundations?
5. Would the overall path still feel meaningful several months in if this were repeated as a pattern?

If those answers are unclear, the expansion is not yet justified as part of the mastery map.
