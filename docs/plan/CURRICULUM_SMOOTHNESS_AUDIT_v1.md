# Curriculum Smoothness Audit v1

Status: ACTIVE AUDIT
Last updated: 2026-05-06

## Purpose

Check whether the course route is not only fully covered, but also sequenced in
a way that can realistically lead a learner from zero to strong play without
conceptual shock, pacing gaps, or hidden curriculum jumps.

This audit asks:

1. is the learning order cognitively coherent
2. are adjacent world transitions smooth
3. does the visible route feel "hand-held" instead of fragmented
4. where is the route already proven vs only planned

## High-Level Verdict

The curriculum route is now strong on sequence logic and early transition
safety.

The active visible spine from W1 through W6 is no longer a loose pile of
topics. It reads as a clean cognitive ladder:

1. understand the table
2. stop overplaying trash
3. understand why seat changes value
4. get the first structured preflop framework
5. understand why bets happen and what a call costs
6. read the board and draw story

This is the right beginner order.

The route is not yet "fully proven end-to-end from zero to elite" in the
production sense, because later worlds are still partly plan-truth rather than
dense authored truth.

So the honest final verdict is:

- **W1-W6 visible route smoothness:** strong and already substantially proven
- **W7-W12 route logic:** now coherent and materially more production-proven
  after W10, W11, and W12 density/seam packets
- **W13-W36 long-term order:** strategically coherent, but still planning truth
  more than production truth

## Proven Smoothness

### 1. Early route order is cognitively correct

The route does not jump too early into strategy abstractions.

It correctly installs:

1. table literacy before discipline
2. discipline before position
3. position before preflop framework
4. preflop framework before bet purpose / price
5. purpose / price before board texture and draws
6. board reading before range thinking

This is exactly the kind of "take the learner by the hand" ordering that
prevents invisible overload.

### 2. Adjacent seam discipline is real, not implied

The active beginner route has explicit seam audit artifacts for:

1. W1 -> W2
2. W2 -> W3
3. W3 -> W4
4. W4 -> W5
5. W5 -> W6

All visible beginner seams currently pass the required checks:

1. concept bridge
2. decision exposure
3. contrast exposure
4. suboptimal literacy
5. vocabulary handoff
6. emotional safety
7. regression lock

That is the strongest evidence that the route is not merely "named well" but
actually transitions in a learner-safe way.

### 3. The route avoids the most dangerous beginner shocks

The current order avoids the three most common bad beginner jumps:

1. asking for strategy before table literacy exists
2. introducing expert language before stable mental anchors exist
3. teaching postflop abstraction before preflop and board reading are stable

### 4. Long-term ladder blocks are strategically coherent

The late ladder is grouped in a sensible escalation:

1. Foundation (W1-W6)
2. Context and range (W7-W11)
3. Mindset bridge (W12)
4. Postflop (W13-W16)
5. Advanced tactics and meta (W17-W21)
6. Solver and theory (W22-W25)
7. Professional tools (W26-W30)
8. Elite (W31-W36)

That block order is strong. It does not show obvious conceptual inversions.

## Remaining Smoothness Risks

### Risk 1. W12 -> W13 is now the next unproven frontier

The W11 -> W12 seam now has a formal artifact with upgraded verdict:

- `docs/plan/SEAM_AUDIT_W11_TO_W12_2026_05_06_v1.md`

Current verdict is `release-playable` after W12 density and regression locks.
The next honest seam frontier is W12 -> W13, where content is still plan-truth.

### Risk 2. W13 and later are still more route-truth than dense learner-proof

The plan is coherent, but the learner path is not yet equally production-proven
beyond the current visible route.

This is the main reason the whole route cannot yet be called perfectly smooth
from zero to capstone in a fully evidenced way.

### Risk 3. Some practical postflop seeds are intentionally delayed

The route still holds these tensions on purpose:

1. basic c-bet framing
2. basic turn/river awareness
3. multiway caution as a larger discipline

This is not a missing-coverage issue.
It is a route-style choice.

The route remains coherent as long as these stay explicitly governed and do not
drift into random earlier tasks.

### Risk 4. Long-term smoothness is structurally planned, not yet production-proven

W13-W36 look coherent on paper, but that is still different from proving that a
real learner will feel no pacing shock there.

Future smoothness proof must come from the same seam-audit discipline already
used in W1-W6.

## What "Ideal Hand-Held Route" Requires Next

To reach the stronger claim "this course can guide a learner from zero to the
end without hidden gaps," the next proof work is not more coverage governance.

It is:

1. keep W13+ honest locked preview until later-volume work is explicitly
   admitted
2. continue the same seam-audit discipline at the next real frontier: `W12 ->
   W13`
3. keep every new world on the same rules: teach first, then decide, then
   recap, then bridge forward

## Bottom Line

Yes, the curriculum can now be checked for smoothness in a meaningful way, and
the answer is mostly positive.

The early route already behaves like a real guided ladder, not a topic pile.

But the honest "perfect from zero to the end" claim is still premature until
the same density and seam-proof discipline extends deeper into W7+ and later
blocks.
