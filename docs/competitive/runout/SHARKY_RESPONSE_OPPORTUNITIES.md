## Top 5 Product Principles To Adopt

1. Package learning as a serious premium system.
2. Make the first session feel authored and confident.
3. Give users a clear daily/session habit.
4. Show skill progress over time.
5. Use poker-native visuals and charts to make abstract concepts concrete.

## Top 5 Things Sharky Must Not Copy

1. Runout assets, backgrounds, videos, icons, chart layouts, typography, or exact wording.
2. Paywall timing if it appears before Sharky has proven first value.
3. Broad tracker/analytics density in the first session.
4. Black-box personalization claims.
5. Any proprietary content structure or implementation details from the APK/bundle.
